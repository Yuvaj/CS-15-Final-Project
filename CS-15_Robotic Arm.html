Code Used for Robotic Arm

Yuvraj Singh – Final Project

Report:

Arduino Code:

This code communicates between the Arduino Uno and the PCA9685 servo driver module via I2C, interprets incoming angle data from Processing, and moves five servos accordingly. Each servo receives a pulse mapped between 150 and 600 (ticks) corresponding to angles between 0° and 180°.

Download Arduino Code

Processing Code:

The Processing program creates a GUI with sliders for each servo, allowing the user to drag and control their angles from the computer. These angles are then transmitted to the Arduino via a serial connection at 9600 baud rate.
