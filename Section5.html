5.1 Testing and Debugging:

  • Unit Testing: Each servo was tested individually for angle range response
  • Integration Testing: Complete control system was evaluated using Processing GUI
  • Debugging: Resolved overheating issues, ensured correct angle parsing, verified PWM signals

5.2 Evaluation:

The system meets its goal of safely controlling servos remotely. Arduino code reliability and user interface responsiveness were confirmed. Though the physical structure wasn't printed, the electronic control system is functional and works. I can confidentially say that if the robotic arm was modelled, this would have 100% worked in picking up my 30 mL beaker.

Improvements: Add printed housing, Add a LCD display to show what angle each joint is at, Add a text box on the side to manually type in desired angle between 0-180.
